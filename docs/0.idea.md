## Github Stars Agent 工作流设计

1. 数据收集模块
    - 通过Github API获取用户所有stars仓库
    - 存储仓库元数据（名称、描述、语言、主题等）
    - 实现增量检测机制（比较最后更新时间）

2. 分类索引模块
    - 基于仓库主题和语言自动分类
    - 生成可搜索的索引结构
    - 支持手动标记和自定义分类

3. 智能推荐模块
    - 基于用户star历史分析兴趣模式
    - 推荐相关或趋势项目
    - 去重和优先级排序

4. 输出生成模块
    - 自动生成分类Markdown文档
    - 可定制的模板系统
    - 支持多种输出格式

5. 定时任务模块
    - 配置定期更新计划
    - 后台自动运行支持
    - 更新通知机制
